Copilot Premium Request 预算管理

User-Level Budgets（用户级预算）· 纯前端调用 GitHub REST API · PAT 仅存于浏览器内存

重要前提：本能力仅支持 Enterprise 账户，是 enterprise-admin 专属功能。不支持组织（org）级、不支持 Copilot Pro/Pro+。不满足前提会导致预算"看起来配置了却不生效"。

前置条件清单（请逐项自查）

  • 使用 Enterprise 账户（Copilot Enterprise / Business / Standalone）。
  • 已在 Settings > AI Controls > Copilot 开启 "Premium request paid overages"。
  • 已移除任何现存的 $0 企业级 premium request 预算（它会优先生效并阻断所有人）。
  • PAT 具备 manage_billing:enterprise scope。
  • 操作者为 Enterprise Owner 或 Billing Manager 角色。

核心概念与优先级

Cost Center
成本中心，用于归集计费的逻辑分组。
Budget（预算）
对 premium request 用量设置的金额上限，达到上限后可阻断后续用量。
Universal 预算（budget_scope: "multi_user_customer"）
面向所有 Copilot 用户的默认预算。每个企业只能有一个 universal 预算，重复创建返回 409。
个人预算（budget_scope: "user"）
面向单个 GitHub 用户的预算。个人预算始终优先于 universal 预算。
系统采用 "剩余额度最小者先阻断（lowest remaining headroom wins）" 逻辑跨层级生效。每位用户的 included premium requests 先消耗，consumed_amount 只反映超出部分的金额。

参数含义

PAT
个人访问令牌，需 manage_billing:enterprise scope。仅保存在浏览器内存中。
enterprise（企业 slug）
企业标识符，用于构造 API 路径。
budget_amount
预算金额（美元）。设为 0 表示默认阻断全部 premium 用量。
budget_scope
multi_user_customer（universal）或 user（个人）。
user
仅 user scope 需要，填 GitHub 用户名（无需数字 ID）。

所有请求固定带头：Accept: application/vnd.github+json、X-GitHub-Api-Version: 2022-11-28、Authorization: Bearer <PAT>。

API curl 速查

以下示例可直接在终端运行。请先设置环境变量：export PAT=<你的令牌>、export ENT=<企业 slug>。所有命令均已带上固定请求头。

① 创建统一预算（Universal，multi_user_customer）

curl -X POST \
  -H "Accept: application/vnd.github+json" \
  -H "X-GitHub-Api-Version: 2022-11-28" \
  -H "Authorization: Bearer $PAT" \
  "https://api.github.com/enterprises/$ENT/settings/billing/budgets" \
  -d '{
    "budget_amount": 50,
    "budget_scope": "multi_user_customer",
    "prevent_further_usage": true,
    "budget_product_sku": "premium_requests",
    "budget_type": "BundlePricing",
    "budget_alerting": { "will_alert": false, "alert_recipients": [] }
  }'

② 创建个人预算（user scope，单个用户）

curl -X POST \
  -H "Accept: application/vnd.github+json" \
  -H "X-GitHub-Api-Version: 2022-11-28" \
  -H "Authorization: Bearer $PAT" \
  "https://api.github.com/enterprises/$ENT/settings/billing/budgets" \
  -d '{
    "budget_amount": 20,
    "budget_scope": "user",
    "user": "octocat",
    "prevent_further_usage": true,
    "budget_product_sku": "premium_requests",
    "budget_type": "BundlePricing",
    "budget_alerting": { "will_alert": false, "alert_recipients": [] }
  }'

③ 列出全部预算

curl \
  -H "Accept: application/vnd.github+json" \
  -H "X-GitHub-Api-Version: 2022-11-28" \
  -H "Authorization: Bearer $PAT" \
  "https://api.github.com/enterprises/$ENT/settings/billing/budgets"

④ 查看指定用户的生效预算（effective_budget）

curl \
  -H "Accept: application/vnd.github+json" \
  -H "X-GitHub-Api-Version: 2022-11-28" \
  -H "Authorization: Bearer $PAT" \
  "https://api.github.com/enterprises/$ENT/settings/billing/budgets?user=octocat&budgetTarget=premium_requests"

⑤ 更新预算（调额，PATCH）

curl -X PATCH \
  -H "Accept: application/vnd.github+json" \
  -H "X-GitHub-Api-Version: 2022-11-28" \
  -H "Authorization: Bearer $PAT" \
  "https://api.github.com/enterprises/$ENT/settings/billing/budgets/$BUDGET_ID" \
  -d '{ "budget_amount": 80 }'

⑥ 删除预算（DELETE）

curl -X DELETE \
  -H "Accept: application/vnd.github+json" \
  -H "X-GitHub-Api-Version: 2022-11-28" \
  -H "Authorization: Bearer $PAT" \
  "https://api.github.com/enterprises/$ENT/settings/billing/budgets/$BUDGET_ID"

公共凭据

① 统一预算（Universal 默认预算）

为所有 Copilot 用户创建默认预算，budget_scope: "multi_user_customer"。每企业仅一个，重复创建返回 409。

② 批量差异化预算（个人预算 · upsert）

逐个为指定用户处理个人预算，budget_scope: "user"。已存在则更新（PATCH），不存在则创建（POST）。支持上传 CSV 或粘贴列表，首行表头 user,budget。

③ Enterprise scope 预算（增量池上限）

用于设置增量池上限，作为 universal 默认预算（budget_scope: "multi_user_customer"）。
